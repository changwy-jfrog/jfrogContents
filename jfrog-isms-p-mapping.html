제품 개요

Artifactory

유니버설 바이너리 저장소 관리자. 모든 패키지 포맷의 아티팩트를 안전하게 저장·배포하고 접근을 통제합니다.

저장소 관리접근통제 감사로그암호화

Curation

오픈소스 패키지 도입 전 정책 기반 자동 차단. 악성·취약 컴포넌트가 개발 환경에 유입되지 않도록 예방합니다.

악성코드 차단취약점 예방 정책 자동화

Advanced Security

SCA·SAST·시크릿 탐지·컨테이너 보안을 통합한 DevSecOps 플랫폼. SBOM 생성 및 위험 우선순위 분석을 제공합니다.

SCA / SAST시크릿 탐지 SBOM취약점 관리

AppTrust

소프트웨어 공급망 무결성 보장. 아티팩트 서명·검증 및 릴리스 신뢰 체계를 통해 변조를 방지합니다.

아티팩트 서명무결성 검증 공급망 보안
1. 관리체계 수립 및 운영

조직의 위험 평가, 자산 식별, 보호대책 선정 단계에서 JFrog 제품군이 기여하는 통제항목

통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
1.2.1 정보자산 식별 Advanced Security의 SBOM(Software Bill of Materials) 자동 생성으로 소프트웨어 구성요소(오픈소스 라이브러리, 컨테이너 이미지, 바이너리)를 체계적으로 식별·목록화합니다. Artifactory는 모든 아티팩트의 메타데이터를 중앙집중 관리하여 정보자산 현황을 실시간 파악할 수 있게 합니다.
Artifactory Advanced Security
1.2.2 현황 및 흐름 분석 Artifactory의 패키지 의존성 그래프와 Advanced Security의 컨텍스트 분석(Contextual Analysis)을 통해 소프트웨어 공급망 흐름을 가시화합니다. 어떤 컴포넌트가 어느 애플리케이션에 포함되어 있는지 상세한 추적이 가능합니다.
Artifactory Advanced Security
1.2.3 위험 평가 Advanced Security는 CVE 기반 CVSS 점수, Exploit 가용성, 실제 취약 함수 호출 여부(Contextual Analysis)를 결합해 위험을 정량적으로 평가합니다. Curation은 패키지 도입 시점에 위험도를 사전 평가하여 리스크를 차단합니다.
Curation Advanced Security
1.2.4 보호대책 선정 위험 평가 결과를 기반으로 Advanced Security가 Fix Version 권고, 대체 패키지 제안을 자동화합니다. Curation 정책을 통해 조직 수준의 보호대책(허용·차단 기준)을 정의하고 일관되게 적용합니다.
Curation Advanced Security
2. 보호대책 요구사항

기술·운영 보안 통제항목과 JFrog 제품군의 구체적 기능 매핑

2.1 정책, 조직, 자산 관리
통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
2.1.3 정보자산 관리 Artifactory는 Docker 이미지, NPM·Maven·PyPI 패키지 등 모든 SW 아티팩트를 중앙 저장소에서 버전·메타데이터와 함께 관리합니다. 불변(Immutable) 저장 및 패키지 잠금(Package Lock) 기능으로 자산의 무결성을 보장합니다. Advanced Security의 SBOM은 자산 목록의 자동 갱신을 지원합니다.
Artifactory Advanced Security
2.5 인증 및 권한관리
통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
2.5.1 사용자 계정 관리 Artifactory는 로컬 계정, LDAP·Active Directory 연동, SAML/SCIM 프로비저닝을 지원하여 사용자 계정의 생성·변경·삭제를 중앙에서 관리합니다. 비활성 계정 자동 비활성화 정책 설정이 가능합니다.
Artifactory
2.5.2 사용자 인증 Artifactory는 SSO(SAML 2.0, OIDC), MFA, API Key, Access Token을 통한 다중 인증 방식을 지원합니다. AppTrust는 아티팩트 배포·소비 시 서명 기반 신원 검증을 추가하여 비인가 배포를 원천 차단합니다.
Artifactory AppTrust
2.5.5 특수 계정 및 권한 관리 Artifactory는 그룹·역할 기반 접근제어(RBAC)로 관리자·서비스 계정의 최소권한 원칙을 적용합니다. CI/CD 파이프라인용 서비스 계정에 대해 범위 제한 토큰(Scoped Token)을 발급하여 권한 남용을 예방합니다.
Artifactory
2.5.6 접근권한 검토 Artifactory의 감사 로그(Audit Log)는 권한 변경 이력을 기록하며, 주기적인 권한 검토를 위한 사용자·그룹별 권한 현황 리포트를 제공합니다.
Artifactory
2.6 접근통제
통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
2.6.1 네트워크 접근 Artifactory는 IP 화이트리스트, 네트워크 정책, VPC 격리(Self-Hosted/Cloud)를 지원합니다. 저장소별·그룹별로 허용 IP 범위를 세분화하여 네트워크 레벨 접근을 통제합니다.
Artifactory
2.6.2 정보시스템 접근 Artifactory의 Permission Target(저장소 단위 권한 설정)으로 사용자·그룹별 읽기·쓰기·배포 권한을 세밀하게 제어합니다. Curation은 정책에 위배되는 패키지 접근을 자동으로 차단합니다.
Artifactory Curation
2.6.3 응용프로그램 접근 Artifactory REST API 및 Web UI 접근에 대해 토큰 기반 인증·만료 정책이 적용됩니다. AppTrust는 서명되지 않은 아티팩트의 배포·사용을 정책적으로 거부하여 응용프로그램 레벨 접근을 통제합니다.
Artifactory AppTrust
2.7 암호화 적용
통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
2.7.1 암호정책 적용 Artifactory는 전송 중(TLS 1.2+) 및 저장 중(AES-256) 암호화를 기본 지원합니다. AppTrust는 아티팩트에 디지털 서명(GPG, Sigstore, x.509 코드서명)을 적용하여 암호화 기반 무결성 보장을 구현합니다. Advanced Security는 시크릿(암호·API키) 노출 여부를 탐지하여 암호정책 위반을 실시간 알립니다.
Artifactory AppTrust Advanced Security
2.7.2 암호키 관리 AppTrust는 서명 키의 생성·등록·폐기·갱신 수명주기를 관리하며, 키 교체 및 신뢰 체인 검증을 지원합니다. Artifactory는 GPG 서명 키를 저장소별로 관리하고 만료 알림을 제공합니다.
Artifactory AppTrust
2.8 정보시스템 도입 및 개발 보안
통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
2.8.1 보안 요구사항 정의 Curation의 정책 정의(Policy-as-Code)를 통해 "CVSSv3 ≥ 7.0 차단", "라이선스 위반 차단", "악성 패키지 즉시 거부" 등 조직별 보안 요구사항을 코드로 정의하고 자동 적용합니다.
Curation Advanced Security
2.8.2 보안 요구사항 검토 및 시험 Advanced Security는 SCA(소프트웨어 구성 분석), SAST(정적 분석), IaC 보안 검사, 컨테이너 이미지 스캔을 CI/CD 파이프라인에 통합하여 코드·빌드 단계에서 자동 보안 시험을 수행합니다.
Advanced Security
2.8.3 시험과 운영 환경 분리 Artifactory의 다중 저장소 환경(Dev / Staging / Prod 저장소 분리)과 프로모션(Promotion) 워크플로우로 검증된 아티팩트만 운영 환경으로 이관됩니다. AppTrust의 서명 검증 게이트가 미승인 아티팩트의 운영 배포를 차단합니다.
Artifactory AppTrust
2.8.5 소프트웨어 개발 보안 Curation은 악성 패키지·취약 라이브러리가 개발 환경 진입 전에 자동 차단됩니다. Advanced Security는 소스 코드 내 보안 취약점·시크릿을 개발 단계에서 조기 탐지하여 "Shift Left Security"를 실현합니다. AppTrust는 소프트웨어 공급망의 각 단계(빌드→배포→실행)에서 서명 체인으로 무결성을 보장합니다.
Curation Advanced Security AppTrust
2.8.6 취약점 관리 Advanced Security는 NVD, JFrog Research, GitHub Advisory 등 복수 데이터베이스를 통합하여 CVE를 탐지하고, Exploit 실현 가능성·실제 코드 호출 여부로 우선순위를 결정합니다. 수정 버전 자동 권고 및 Jira·Slack 연동으로 취약점 조치 프로세스를 자동화합니다.
Curation Advanced Security
2.9 시스템 및 서비스 운영관리
통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
2.9.1 변경관리 Artifactory는 아티팩트 불변성(Immutability), 버전 이력, 메타데이터 변경 이력을 기록합니다. AppTrust의 서명 체인은 어느 파이프라인에서 언제 빌드·서명되었는지 추적 가능한 감사 증적을 생성합니다.
Artifactory AppTrust
2.9.4 로그 및 접속기록 관리 Artifactory는 모든 API 호출·UI 접속·아티팩트 업로드·다운로드에 대한 감사 로그를 구조화된 포맷으로 기록합니다. Splunk, Datadog, ELK 등 SIEM 시스템과 연동하여 중앙화된 로그 관리가 가능합니다.
Artifactory
2.9.5 로그 및 접속기록 점검 Artifactory 감사 로그는 사용자별·저장소별·시간대별 접근 패턴 분석을 지원하며, Advanced Security의 이상행위 탐지와 연계하여 비정상 접속을 자동 감지합니다.
Artifactory Advanced Security
2.10 시스템 및 서비스 보안관리
통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
2.10.2 클라우드 보안 JFrog Cloud(SaaS) 환경은 SOC 2 Type II, ISO 27001 인증을 보유하며, 멀티 클라우드(AWS·GCP·Azure)에서 데이터 상주 지역 선택이 가능합니다. VPC 격리·PrivateLink 연결로 공용 인터넷 노출 없이 운영할 수 있습니다.
Artifactory
2.10.5 정보전송 보안 Artifactory는 모든 전송 경로에서 TLS 1.2/1.3 강제 적용합니다. AppTrust는 아티팩트 배포 시 서명 검증을 통해 전송 중 변조 여부를 탐지하며, 미서명 아티팩트의 배포를 정책적으로 거부합니다.
Artifactory AppTrust
2.10.8 패치 관리 Curation은 취약 버전 패키지의 신규 도입을 정책 기반으로 자동 차단하고 안전한 버전을 권고합니다. Advanced Security는 이미 배포된 아티팩트에 대한 취약 컴포넌트 버전을 지속 모니터링하여 패치 필요 항목을 우선순위화합니다.
Curation Advanced Security
2.10.9 악성코드 통제 Curation은 악성 패키지(Malicious Package), 타이포스쿼팅, 의존성 혼동 공격을 자동 탐지하고 차단합니다. Advanced Security는 바이너리 내 악성 코드 패턴 및 알려진 악성 해시를 실시간으로 스캔합니다.
Curation Advanced Security
2.11 사고 예방 및 대응
통제항목 통제항목명 JFrog 제품 기여 내용 관련 제품
2.11.1 사고 예방 및 대응체계 구축 Curation의 사전 차단과 Advanced Security의 실시간 스캔이 결합된 다층 예방 체계를 구축합니다. AppTrust는 공급망 공격(SolarWinds류) 예방을 위한 아티팩트 신뢰 체인을 제공합니다. Jira·Slack·PagerDuty 연동으로 취약점 발견 시 자동 티켓 생성·알림 발송이 가능합니다.
Curation Advanced Security AppTrust
2.11.2 취약점 점검 및 조치 Advanced Security는 저장소 전체를 지속적으로 스캔하며, 새로운 CVE가 발표되면 기존 아티팩트에 대해 자동 재평가합니다. SBOM 기반으로 영향 받는 애플리케이션 범위를 신속하게 식별하고, 수정 버전 업그레이드 PR을 자동 생성합니다.
Advanced Security
2.11.3 이상행위 분석 및 모니터링 Artifactory 감사 로그를 기반으로 비정상 다운로드 패턴, 새벽 시간대 접속, 대용량 데이터 유출 시도 등을 SIEM 연동을 통해 실시간 탐지합니다. Advanced Security의 컨텍스트 분석은 취약 함수가 실제 호출되는지 분석하여 오탐을 줄이고 실질적 위협에 집중합니다.
Artifactory Advanced Security
